# Javascript 30 Days Challenge

This repository was created to develop the projects from the **JavaScript30** series by Wes Bos by **rebuilding each project from scratch over 30 days**.

The main goal of this challenge is to strengthen my understanding of **Vanilla JavaScript**, improve DOM manipulation skills, and practice JavaScript through real-world mini projects without using any frameworks or libraries.
